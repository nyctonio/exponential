# Admin

The admin user types are : **Company, Master, Broker and Sub-broker**.

#### Access and rights

**Admin has the following rights :**

1. User Creation and Updation.
2. Change the user's trade settings.
3. Manage, delete trades and positions of the users.
4. User access management.

#### User Creation and Updation

This is the module for creating creating and updating the users.
The hierarchy of the users goes like : **Company, Master, Broker, Sub-broker and Client.**

Company can create master and client, Master can create Broker and client, Broker can create Sub-broker and Client and Sub-broker can create only a Client.
